本章節講到迴圈(while, do while, for)

## while
```
while (條件判斷) { 
    敘述 
} 
```

當條件判斷為true時，會繼續迴圈，直到條件判斷為false時，則跳出迴圈。 

我們來看一個例子

```
x = 6;
i = 1;

while (i < x) { 
     printf("%d\n", i);
     i++;
} 
```

此段程式碼輸出結果為

```
1
2
3
4
5
```

因為x值為5，i初始值為1，首先進行判斷(1 < 5)輸出為true，所以會進行迴圈裡的敘述
```
printf("%d\n", i); //印出i值
i++; //i + 1
```

然後回去繼續判斷
```
    //此時i為2，因為在第一次迴圈時i+1
    while (i < x) { 
        printf("%d\n", i);
        i++;
    }
```

此迴圈會一直運作直到 i >= x輸出為false跳出迴圈

## do...while

```
do {
    敘述
} while (條件判斷);
```

do...while基本上與while一樣，不同的是他是先執行敘述再進行條件判斷

```
x = 5;
i = 1;

do {
    printf("%d", i);
    i++;
} while (i < x);
```

此段程式碼輸出結果為
```
1
2
3
4
```
因為當i = 4，執行了i++後判斷為false所以只輸出到4

---

## for

```
for (初始值, 判斷式, 遞增、遞減...) {
    敘述
}
```

for迴圈和while迴圈的差異在於，for迴圈可以寫出初始值、遞增、遞減...
```
for (int i = 1;, i < 6;, i++) {
    printf("%d, ", i);
}
```
跟while比for較簡單易讀
